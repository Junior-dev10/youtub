# Projet Calculatrice Web Interactive

Une application de calculatrice moderne, responsive et intuitive développée en **HTML5**, **CSS3** et **JavaScript (ES6+)**.

## 🚀 Fonctionnalités

- **Opérations Arithmétiques** : Support complet des calculs de base (addition, soustraction, multiplication, division).
- **Gestion de la Saisie** :
  - Touche `C` pour réinitialiser complètement l'affichage.
  - Touche `DEL` pour supprimer le dernier caractère saisi.
- **Interface Moderne** : Design sombre (Dark Mode) avec des dégradés radiaux et des ombres portées pour une profondeur visuelle.
- **Design Responsive** : Mise en page adaptative utilisant **CSS Flexbox** pour le centrage et **CSS Grid** pour l'organisation des touches.

## 🛠️ Technologies utilisées

- **HTML5** : Structure de l'interface.
- **CSS3** : Stylisation avancée, Flexbox, Grid Layout et Gradients.
- **JavaScript** : Manipulation du DOM, gestion des événements et logique de calcul via `eval()`.

## 📂 Installation

1. Clonez ce dépôt sur votre machine locale.
2. Ouvrez le fichier `index.html` dans n'importe quel navigateur moderne.
3. Commencez à calculer !
